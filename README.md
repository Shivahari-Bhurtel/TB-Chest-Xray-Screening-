# TB-Chest-Xray-Screening-
# Tuberculosis Detection from Chest X-ray Images

This project uses deep learning to detect **Tuberculosis (TB)** from chest X-ray images.

Two models were developed and compared:

- CNN (Built from Scratch)
- MobileNetV2 (Transfer Learning)

The models were evaluated using Accuracy, Precision, Recall, F1 Score, and Confusion Matrix. Based on the evaluation results, **MobileNetV2** achieved better performance and was selected for deployment using **Gradio**.

## Technologies Used

- Python
- TensorFlow / Keras
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Gradio
